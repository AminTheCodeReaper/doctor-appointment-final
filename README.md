# 🩺 Doctor Appointment System

A modern, full-stack web application for online doctor appointment booking, built with Next.js, TypeScript, Tailwind CSS, and MongoDB.

## ✨ Features

- 🔐 **OTP Authentication** – Login with phone number and one-time verification code
- 👨‍⚕️ **Doctor List** – Search and filter doctors by name or specialty
- 📋 **Doctor Details** – View doctor information, working hours, address, and ratings
- 📅 **Appointment Booking** – Select date and time, then confirm your visit
- 📋 **Appointment Management** – View, cancel, or pay for your appointments
- 👤 **User Profile** – Edit personal information
- 🧪 **Health Test** – Quick health assessment with results
- 📱 **Fully Responsive** – Optimized for mobile, tablet, and desktop

## 🛠️ Tech Stack

- **Frontend:** Next.js 15 (App Router), TypeScript, Tailwind CSS
- **Backend:** Next.js API Routes, MongoDB, Mongoose
- **Auth:** JWT, OTP (SMS.ir)
- **Tools:** React Hook Form, Lucide Icons

## 🚀 Quick Start

### Prerequisites
- Node.js 18+
- MongoDB (local or Atlas)
- Git

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/AminTheCodeReaper/doctor-appointment-final.git
   cd doctor-appointment
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Seed the database:
   ```bash
   npx tsx scripts/seed.ts
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

5. Open [http://localhost:3000](http://localhost:3000)
